Le mie esperienze principali di PCTO

In questa pagina raccolgo le esperienze più importanti svolte durante il percorso scolastico, concentrandomi sui progetti PCTO del terzo e quarto anno.

3

Terzo anno - Electrolink

Hardware e software

During my FSL experience at Electrolink, I worked on both hardware and software activities related to radio transmitters and DAB systems. I participated in debugging radiofrequency transmitters, assembling coaxial cables, soldering electronic boards and testing DAB transmitters.

On the software side, I contributed to the development of transmitter web interfaces using HTML, JavaScript and web logic. I also worked on graphical interface design for transmitter manuals and on the programming of a 5kW transmitter website.

This experience allowed me to improve my technical skills in electronics, web programming, debugging and teamwork, while also learning the theoretical functioning of DAB transmission systems.

4

Quarto anno - B4Next

Web development

During my FSL experience at B4Next, I worked on web development projects using HTML, CSS and JavaScript with Visual Studio Code. I first recreated a Serie A website and implemented SportsDB APIs through the JavaScript fetch() function to display dynamic online data.

Later, I focused on Spotify APIs, studying token authentication and API requests. I developed a music-themed website capable of searching songs, albums and playlists using Spotify APIs.

This experience improved my frontend development skills and my knowledge of JavaScript logic, API integration and dynamic web applications.

Competenze acquisite

  • Frontend development con HTML, CSS e JavaScript
  • Integrazione di API e dati dinamici
  • Debugging hardware e software
  • Nozioni su trasmettitori radio e sistemi DAB
  • Teamwork e organizzazione del lavoro

Riflessione finale

Le esperienze presso Electrolink e B4Next mi hanno permesso di collegare elettronica, programmazione web e lavoro su progetti reali. Ho potuto vedere sia il lato tecnico dei sistemi hardware sia quello dello sviluppo frontend, migliorando autonomia, precisione e capacità di problem solving.
