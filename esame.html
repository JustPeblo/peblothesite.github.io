My main PCTO experiences

This page presents the most important experiences from my school path, focusing on the PCTO projects I completed during my third and fourth year.

3

Electrolink

Third year

During my PCTO experience at Electrolink, I worked on both hardware and software activities related to radio transmitters and DAB systems. I participated in debugging radiofrequency transmitters, assembling coaxial cables, soldering electronic boards and testing DAB transmitters.

On the software side, I contributed to the development of transmitter web interfaces using HTML, JavaScript and web logic. I also worked on graphical interface design for transmitter manuals and on the programming of a 5kW transmitter website.

This experience allowed me to improve my technical skills in electronics, web programming, debugging and teamwork, while also learning the theoretical functioning of DAB transmission systems.

4

B4Next

Fourth year

During my PCTO experience at B4Next, I worked on web development projects using HTML, CSS and JavaScript with Visual Studio Code. I first recreated a Serie A website and implemented SportsDB APIs through the JavaScript fetch() function to display dynamic online data.

Later, I focused on Spotify APIs, studying token authentication and API requests. I developed a music-themed website capable of searching songs, albums and playlists using Spotify APIs.

This experience improved my frontend development skills and my knowledge of JavaScript logic, API integration and dynamic web applications.

Skills acquired

  • Frontend development with HTML, CSS and JavaScript
  • API integration and dynamic data handling
  • Hardware and software debugging
  • Basic knowledge of radio transmitters and DAB systems
  • Teamwork and work organization

Final reflection

My experiences at Electrolink and B4Next allowed me to connect electronics, web programming and real project work. Working in both environments helped me understand the difference between hardware-oriented tasks and software development, and it made me realize that I prefer programming because I enjoy building logic, interfaces and dynamic web applications more than working directly on hardware.
