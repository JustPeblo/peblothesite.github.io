Percorso personale

Formazione scuola lavoro

Un percorso tra elettronica, sviluppo web e progetti reali: dal lavoro sui trasmettitori radio alla creazione di interfacce e applicazioni dinamiche.

Esperienze principali

Dal laboratorio al frontend

3 anno Electrolink

Trasmettitori radio, DAB e interfacce web

During my FSL experience at Electrolink, I worked on hardware and software activities related to radio transmitters and DAB systems. I participated in debugging radiofrequency transmitters, assembling coaxial cables, soldering electronic boards and testing DAB transmitters.

On the software side, I contributed to transmitter web interfaces using HTML, JavaScript and interface logic, also working on graphical design for manuals and on a 5kW transmitter website.

  • Debugging RF
  • Saldatura
  • DAB
  • HTML
  • JavaScript
4 anno B4Next

Siti dinamici, API e ricerca musicale

During my FSL experience at B4Next, I worked on web development projects using HTML, CSS and JavaScript with Visual Studio Code. I recreated a Serie A website and used SportsDB APIs with fetch() to display dynamic online data.

Later, I focused on Spotify APIs, studying token authentication and API requests, then developing a music-themed website capable of searching songs, albums and playlists.

  • HTML
  • CSS
  • JavaScript
  • API
  • Fetch
